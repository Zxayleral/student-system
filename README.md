# student-system
Simple Student Information System with Graphical User Interface

This project is made from Python 3.10 <= @latest

## Dependencies
<ul>
	<li>tkinter</li>
	<li>csv</li>
</ul>


## To run this program, just clone the app and run src/launcher.py
```bash

# in linux
python3 src/launcher.py

# in windows
py src/launcher.py
```
